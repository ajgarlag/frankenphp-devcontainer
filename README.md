# FrankenPHP Devcontainer

A [Devcontainer](https://containers.dev/) runtime for [PHP](https://php.net),
with [FrankenPHP](https://frankenphp.dev) and [Caddy](https://caddyserver.com/) inside!

Based on the [Symfony Docker](https://github.com/dunglas/symfony-docker) project.
